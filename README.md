# 포인트 파머를 위한 시각화 툴
## 실행 방법
1. `backend` 디렉토리에서 터미널 열고 `npm install` 후 `npm start`
2. `frontend/index.html` 우클릭 후 Open with Live Server
## 제작 계기
- 포인트 파밍을 목적으로 하는 유저가 있으며, 많아질 것이라고 생각함
- 잠재적인 생태계 참여자들의 진입 장벽을 낮추기 위해 제작함
- 또한, 효율적으로 파밍하려는 사용자에게 최적의 경험을 제공할 수 있음
- 다양한 프로젝트(특히 소규모 또는 신생)에 대한 관심도가 올라갈 것이라고 예상하고, 이는 하이퍼리퀴드 생태계에 선순환이 될 것이라 생각함
## 참가 후기
- 인생 첫 해커톤인데 좋은 경험이었음
- 시간적인 여유가 생긴다면 privy를 사용한 지갑 로그인 기능과 검증된 defi 프로젝트 주소 라벨링을 진행하고 싶음
