# Point Farmer Visualization Tool
## How to Run
1. Open terminal in `backend` directory, run `npm install` then `npm start`
2. Right-click on `frontend/index.html` and select "Open with Live Server"
## Development Background
- There are users whose purpose is point farming, and I believe this number will increase
- Created to lower the entry barrier for potential ecosystem participants
- Can provide optimal experience for users who want to farm efficiently
- I expect interest in various projects (especially small-scale or emerging ones) to increase, which I believe will create a positive cycle for the Hyperliquid ecosystem
